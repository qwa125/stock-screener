# 股票信号分析 App - 设计思路

## 产品定位
简洁的股票查询工具，输入股票代码/名称，基于同花顺指标公式自动分析买卖信号。

## 技术方案

### 数据源
- **东方财富 API**：免费获取股票日K线数据（前复权）
- 搜索接口：股票名称→代码转换

### 核心架构
```
用户输入股票代码/名称
  → 后端搜索/转换股票代码
  → 获取日K线历史数据 (500条)
  → 公式计算引擎 (TS实现同花顺函数)
    → 白△公式计算
    → 白◇公式计算  
    → 白☆公式计算
    → ☆★公式计算
  → 组合判断逻辑 → 输出买卖信号
  → 返回结果给前端展示
```

### 公式引擎设计
在后端实现同花顺核心函数：
- **基础统计**：MA, EMA, LLV, HHV, STD, SUM, REF
- **逻辑判断**：CROSS, COUNT, BARSLAST, IF, MAX, MIN, ABS
- **支持批量向量计算**（对数组操作）

### 组合规则
按用户定义的规则判断买卖信号：
- 白消第4-6天的买入组合
- 白消第6天以上的买入组合
- 白布状态下的买入条件
- 白消/白布区域的卖出信号

### 前端设计
- 简洁搜索框 + 搜索按钮
- 结果卡片展示：股票名称/代码、当前位置区域、趋势状态、买卖信号
- 信号详情：买入原因、卖出原因拆解

## 配色
- 主色调：深蓝 + 金色（金融风格）
- 买入信号：红色/红色系
- 卖出信号：绿色
- 持有信号：橙色
- 风险信号：紫色
